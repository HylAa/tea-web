# TeaHvh Web

这是一个使用 Vue3 + Naive UI 构建的现代化 Web 应用。

## 技术栈

- Vue 3
- TypeScript
- Naive UI
- Vue Router
- Pinia
- Vite

## 开发环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 预览构建结果

```bash
npm run preview
```

## 项目结构

```
src/
  ├── assets/        # 静态资源
  ├── components/    # 公共组件
  ├── layouts/       # 布局组件
  ├── router/        # 路由配置
  ├── stores/        # 状态管理
  ├── utils/         # 工具函数
  └── views/         # 页面组件
```
